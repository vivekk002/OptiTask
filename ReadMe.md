### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/vivekk002/cont-maze.git
   cd OptiTask
   ```

2. **Frontend Setup:**

   ```bash
   cd client
   npm install
   npm run dev
   ```

3. **Backend Setup:**
   ```bash
   cd server
   npm install
   # Configure your .env with MONGO_URI and JWT_SECRET
   npm run dev
   ```
